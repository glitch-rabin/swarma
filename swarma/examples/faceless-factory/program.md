# Faceless Factory

## Mission

Produce faceless TikTok/YouTube Shorts at scale with systematic retention optimization. Every batch is an experiment -- test hooks, formats, lengths, and visual styles against real performance data. Target: 30+ videos/week with iterating quality.

## How This Actually Works in 2026

The faceless pipeline is fully automatable: topic queue -> LLM script -> ElevenLabs voice -> Leonardo AI/Flux images -> Creatomate/Shotstack assembly -> platform API publish. Zero human touch per video. The entire stack costs $50-300/month.

The catch: YouTube suspended thousands of faceless AI channels in early 2026 under inauthentic content policy. Mass-produced slop gets flagged. Survivors add genuine research, unique angles, or curated data. This squad's job is to be in the survivor category -- every script needs a real insight, not regurgitated facts.

Revenue by niche: Personal Finance ($12-20 CPM), Tech/AI ($8-15 CPM), True Crime ($6-12 CPM), Education ($4-8 CPM). Top earners pull $5K-80K/month at scale.

## Experiment Patterns

1. **Hook format A/B**: question hook vs shocking stat vs contrarian claim -- measure CTR
2. **Voice clone vs stock voice**: same script, different voice -- measure AVD
3. **Visual style**: stock footage vs AI-generated imagery vs mixed -- measure retention curve
4. **Length optimization**: 30s vs 60s vs 90s -- measure completion rate and RPM
5. **Niche crossover**: same topic reframed for finance vs tech vs general audience
6. **Posting cadence**: 1/day vs 3/day -- measure algorithm boost vs diminishing returns
7. **Platform split**: same content on YouTube Shorts vs TikTok vs Reels -- measure RPM per platform
8. **Trending audio patterns**: original audio vs trending sounds -- measure initial distribution

## Metrics

- **Primary**: average_view_duration (target: >50% for Shorts)
- **Secondary**: CTR (target: >5%), publish_frequency (target: 1-3/day)
- **Revenue**: RPM by niche, views/day volume
- **Quality gate**: if AVD drops below 40%, pause publishing and diagnose scripts

## Constraints

- Every script must contain at least one specific data point or insight not in the first page of Google results
- Hook-first: first 3 seconds determine everything. Script writer leads with the hook, not the topic.
- Batch production: scripts generated in batches of 10+, not one at a time
- No generic "did you know" energy. Practitioner framing or unique data only.
