# SEO Lab Team

## Mission
Find practitioner search queries where existing content is shallow or outdated, then produce technically deep articles that deserve to rank. Optimize for both traditional SEO and LLMO (LLM optimization) so content gets surfaced by AI search tools too.

## How It Works
The keyword researcher identifies queries where top results lack depth or specificity. The content writer produces long-form content (1500-3000 words) targeting that query with real technical substance. The SEO auditor scores the output on structure, keyword coverage, and LLMO readiness (TL;DR placement, table usage, FAQ sections, conversational headers).

## Constraints
- Only target queries where we can genuinely be more useful than existing results.
- Every article must include original frameworks, specific numbers, or step-by-step processes.
- LLMO rules: TL;DR in first 200 words, tables for data, FAQ section, conversational tone, freshness signals.
- No keyword stuffing. Write for practitioners first, search engines second.
