# Community Engine

## Mission

Build community-led growth systems with measurable engagement loops. XP/quest gamification, cross-platform funnels, UGC amplification, and ambassador programs -- all optimized against DAU/MAU ratio (target: >20%) and conversion rate (member -> customer).

## How This Actually Works in 2026

The stack: onboarding automation (welcome DM -> role selection -> channel routing) -> XP + leveling system (messages, reactions, events, quests) -> quest system (on-chain + off-chain tasks via Galxe/Zealy) -> engagement loops (daily check-ins, weekly challenges, leaderboards) -> cross-platform integration (Twitter actions trigger Discord rewards) -> funnel automation (Telegram/Discord as lead capture -> nurture -> conversion).

Personalized onboarding increases engagement by 40%. The game layer (XP, streaks, leaderboards) creates habitual return patterns. Ambassador programs turn power users into growth agents.

## Experiment Patterns

1. **Onboarding flow**: instant access vs guided tour vs quiz-based routing -- measure D7 retention
2. **XP decay vs no decay**: losing XP for inactivity -- drives re-engagement or churn?
3. **Quest reward type**: XP only vs NFT vs discount vs exclusive access -- measure completion rate
4. **Streak mechanics**: daily login bonus vs weekly challenge -- measure DAU consistency
5. **Channel architecture**: few broad vs many niche channels -- measure message volume and satisfaction
6. **Cross-platform quests**: Twitter + Discord combined vs Discord-only -- measure growth rate
7. **Token gating threshold**: low barrier vs high barrier for premium channel -- measure conversion + quality

## Metrics

- **Primary**: dau_mau_ratio (target: >20%)
- **Engagement**: message_volume, quest_completion_rate (target: >30%)
- **Retention**: D1/D7/D30 retention, role_progression_rate
- **Growth**: referral_rate, viral_coefficient
- **Conversion**: community_member_to_customer_rate

## Constraints

- Gamification must serve engagement, not exploit it. XP systems that create obligation = churn.
- Quality > quantity for messages. 50 meaningful conversations > 500 "gm" messages.
- Moderation is non-negotiable. One spam wave can destroy community trust overnight.
- Cross-platform quests must not violate platform ToS (especially LinkedIn/Twitter)
- Token gating must be accessible. High barriers create exclusivity but kill growth.
